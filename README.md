# SQLZoo Solutions

My solutions to SQLZoo tutorials and quizzes, written in MySQL.
Each folder contains the tutorial solutions, quiz answers, and any additional practice questions.

## Concepts Covered

**01 - Select Basics**
- SELECT, FROM, WHERE
- BETWEEN, IN, LIKE
- Basic arithmetic on columns

**02 - Select from Nobel**
- Filtering with WHERE, AND, OR
- Pattern matching with LIKE
- Subqueries and NOT IN
- GROUP BY and HAVING
- ORDER BY with custom sorting

## Tools
- MySQL
- PopSQL
